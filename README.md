## Support Platform
Linux:
```
- amd64 ( x68_64 )
- arm64-v8a ( aarch64 )
- s390x
- Riscv64
```
## Cài Đặt Bằng Script [ Đọc lưu ý phía dưới ]
```
Tắt Unikey trước khi chạy Script. Tránh trình trạng Sai Mật Khẩu
```
